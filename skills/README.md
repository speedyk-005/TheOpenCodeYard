# contribute-scrapt

Handle full deployment of scrap code or prototypes to The Open CodeYard.

## Installation

Place this skill under `.agents/skills/contribute-scrapt/` in your pi agent directory.

## Usage

See `SKILL.md` for full instructions.

**Quick trigger examples:**
- "Add this scrap" + paste code
- "Deploy this prototype at ~/projects/thing/"
- "I want to contribute this to the yard"
