# 🏗️ The Yards

This is where the actual code lives. Organized into zones based on how cooked the idea is:

| Zone | Vibe |
|------|------|
| **[scraps/](scraps/)** | Random leftovers, half-finished thoughts, notes. Low ceremony. |
| **[prototypes/](prototypes/)** | Intentional experiments, minimal implementations, "will this work?" builds. |

Each yard contains folders named after contributors, and inside those, their projects.

Browse around. Something might be useful.
